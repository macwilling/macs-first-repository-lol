Now, I'm going to use CSS to demonstrate three different FlexBox properties.

Below, you will find three black boxes, each containing five smaller boxes. The black box is the "parent" content block, and the smaller boxes are individual "child" content blocks.

The black boxes will adjust to the size of your browser, not using FlexBox, but using a simple width property in CSS: #container1 {width: 100%}

This line of code tells container1 to expand its across the entire content block (i.e., from left margin to right margin).

The template for the following demonstrations was created using this HTML and CSS code:

            
                <section id="container1">
                    <div style="background-color: #6a040f"></div>
                    <div style="background-color: #9d0208"></div>
                    <div style="background-color: #d00000"></div>
                    <div style="background-color: #dc2f02"></div>
                    <div style="background-color: #e85d04"></div>
                </section>
            
        
            
    #container1 {background-color: black; width: 100%; height: 400px;}
    
    #container1 div {width: 100px; height: 100px;}

Now, we can use different FlexBox properties to easily change how the child boxes will be layed out within the parent box. In this example, we will look at three variants of the justify-content property.

Spaced Evenly 

            
                    display: flex;
                    justify-content: space-evenly;
                    align-items: center;

Spaced Around 

            
                    display: flex;
                    justify-content: space-around;
                    align-items: center;

Spaced Between 

            
                    display: flex;
                    justify-content: space-between;
                    align-items: center;
